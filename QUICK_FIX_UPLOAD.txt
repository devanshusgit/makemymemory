================================================================================
                    QUICK FIX: PRODUCT UPLOAD NOT WORKING
================================================================================

ISSUE: Can't upload photos or add products in admin panel

SOLUTION: 99% of the time, this is fixed by restarting the dev server

================================================================================
STEP 1: STOP THE DEV SERVER
================================================================================

In your terminal where npm run dev is running:
  Press: Ctrl+C

Wait for it to stop completely.

================================================================================
STEP 2: RESTART THE DEV SERVER
================================================================================

In the same terminal:
  npm run dev

Wait for it to say "ready - started server on 0.0.0.0:3001"

================================================================================
STEP 3: CLEAR BROWSER CACHE
================================================================================

In your browser:
  Windows/Linux: Ctrl+Shift+R
  Mac: Cmd+Shift+R

This does a hard refresh and clears the cache.

================================================================================
STEP 4: LOGIN AS ADMIN
================================================================================

Go to: http://localhost:3001/admin/login

Enter:
  Password: admin123456

Click: Login

================================================================================
STEP 5: TRY UPLOADING AGAIN
================================================================================

Go to: http://localhost:3001/admin/products

Click: "+ Add Product"

Fill in:
  - Name: Test Product
  - Description: Test description
  - Price: 999
  - Category: foil-imprints

Upload a photo:
  - Click the upload area
  - Select an image file
  - Wait for it to upload

Click: Save Product

================================================================================
IF IT STILL DOESN'T WORK
================================================================================

1. Check browser console (F12 → Console)
   - Look for red error messages
   - Screenshot the error

2. Check .env.local file
   - Make sure it has these lines:
     NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=dwhj8vjqr
     CLOUDINARY_API_KEY=624865681549231
     CLOUDINARY_API_SECRET=VkftQL4V8Nu9jcmuzfarU7KO5CI
     ADMIN_PASSWORD=admin123456

3. If .env.local is missing these, add them and restart dev server

4. Try a different browser
   - Chrome, Firefox, Safari, Edge

5. Try a smaller image file
   - Use a file under 1MB
   - Try JPG format
